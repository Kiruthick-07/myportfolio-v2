# Portfolio Website

A personal portfolio website built with React and Vite, designed to showcase skills, projects, and experience.

## Features

- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Interactive UI**:
  - Typewriter effect for roles.
  - Hover animations for navigation links and social icons.
  - Smooth scrolling.
- **Modern Styling**: Clean and minimalist aesthetic using CSS.

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: CSS3
- **Tooling**: ESLint

## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Kiruthick-07/myportfolio-v2.git
   ```
2. Navigate to the project directory
   ```sh
   cd portfolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```

### Usage

Run the development server:
```sh
npm run dev
```

Build for production:
```sh
npm run build
```
